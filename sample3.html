validating number fields

Input fields with attribute type="number" or type="text" can be validated by adding data-type="number" to that fields.

Input fields below are validated only by adding the following required attributes to the input field:

data-init - used on form field for initializing message display

data-type="number" - used on input field for changing accepted data to number

My Form



Note that:
  1. When the form field is not initialized with data-init, all input elements with type='number' may not display message when string is added intead of number. They however, will be validated internally. To solve this issue, consider using an input <input type="text"/> then set the data type to number i.e <input type="text" data-type="number"/>
  2. All attributes discussed on this page may be used in other samples but will not be explained.
